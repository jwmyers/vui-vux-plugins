# Color System

## Player Colors

| Player | Hex       | RGB            | Purpose                    |
| ------ | --------- | -------------- | -------------------------- |
| Red    | `#FF2244` | (255, 34, 68)  | Offensive/attacking routes |
| Blue   | `#44BBFF` | (68, 187, 255) | Defensive/security routes  |

## Path Colors

| Path Type       | Hex       | Usage                     |
| --------------- | --------- | ------------------------- |
| Red paths       | `#FF2244` | Red player only           |
| Blue paths      | `#44BBFF` | Blue player only          |
| Shared (Purple) | `#BB88FF` | Both players can traverse |

## Background Colors

| Element          | Hex       | RGB                         |
| ---------------- | --------- | --------------------------- |
| Board Background | `#0D0F14` | (13, 15, 20)                |
| Tile Background  | `#151820` | (21, 24, 32)                |
| Grid Lines       | `#222630` | (34, 38, 48) at 50% opacity |

## UI Elements

| Element                | Color            |
| ---------------------- | ---------------- |
| Grid Border (Firewall) | Purple `#BB88FF` |
| Blue Zone Border       | Blue `#44BBFF`   |
| Red Zone Border        | Red `#FF2244`    |

## Accessibility

- High contrast between paths and background
- Distinct in color-blind simulations
- Red appears brownish but remains distinct
- Blue remains clearly visible
- Purple maintains distinct identity
